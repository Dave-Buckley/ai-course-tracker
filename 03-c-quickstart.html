C Quickstart

8 focus blocks ~4 hours total Phase 2 prep Reference: CS50x Week 1

What is C, and why are we touching it? C is an older, lower-level language than Python. Where Python hides things like memory and types, C exposes them — you have to think about pointers, allocate memory by hand, and pick exact integer sizes. CS50x uses C as a teaching language for those reasons. After this worksheet you'll have enough C to follow the lectures and you'll appreciate Python a LOT more.

No huge ambitions here — just enough C to follow CS50x weeks 1-5. The mini-project is a Caesar cipher, a CS50 classic.

3. Compile and run

What: turn the .c text file into an .exe binary (compile), then run the binary.

In VS Code's terminal (View → Terminal):

gcc hello.c -o hello
./hello

-o hello names the output file hello (otherwise it'd be a.exe). ./hello runs it from the current folder.

4. Compile with warnings on

Why: warnings catch bugs before they bite. Always compile with them enabled.

Recompile:

gcc -Wall -Wextra hello.c -o hello
Worked? No warnings printed (clean code). If warnings appear, fix them.

Types and printf format specifiers

1. 02-types.c — int, float, double, char

What: C makes you declare a type for every variable, unlike Python.

Create 02-types.c. Type:

#include <stdio.h>

int main(void)
{
    int age = 32;
    float ratio = 3.14f;
    double precise = 3.141592653589793;
    char letter = 'A';
    printf("int: %d\n", age);
    printf("float: %f\n", ratio);
    printf("double: %lf\n", precise);
    printf("char: %c\n", letter);
    return 0;
}

Compile + run:

gcc -Wall 02-types.c -o 02-types
./02-types
Worked? Four lines printed with each value.

3. 03-overflow.c — int overflow

Why: C ints are fixed-width. Overflow is silent and produces wrong answers.

Create. Type:

#include <stdio.h>

int main(void)
{
    int x = 2147483647;   // max 32-bit int
    x = x + 1;
    printf("%d\n", x);
    return 0;
}

Compile + run.

Worked? Prints a large negative number — the int wrapped around. Python doesn't do this; C does.

4. 04-truncation.c — int division

Compile + run.

Worked? int division prints 3 (truncated), float division prints 3.5.

Control flow + scanf

1. 05-conditionals.c — read a number, classify it

What: scanf reads from the user. Note the & before the variable — that's the address-of operator (more in Block 6).

Create. Type:

#include <stdio.h>

int main(void)
{
    int n;
    printf("Type a number: ");
    scanf("%d", &n);
    if (n < 0)       printf("negative\n");
    else if (n == 0) printf("zero\n");
    else             printf("positive\n");
    return 0;
}

Compile + run. Type a number when prompted.

Worked? Classifies whatever you typed.

4. 08-switch.c — vowel or consonant

Create. Type:

#include <stdio.h>

int main(void)
{
    char c;
    printf("Type a letter: ");
    scanf(" %c", &c);   // space before %c skips whitespace
    switch (c) {
        case 'a': case 'e': case 'i': case 'o': case 'u':
        case 'A': case 'E': case 'I': case 'O': case 'U':
            printf("vowel\n"); break;
        default:
            printf("consonant\n");
    }
    return 0;
}

Compile + run.

Worked? Correctly classifies vowels and consonants.

2. 10-prototype.c — forward declaration

Why: C reads files top-to-bottom. If you call a function before it's defined, you must declare its signature (a "prototype") at the top.

Create. Type:

#include <stdio.h>

int square(int);   // prototype

int main(void)
{
    printf("%d\n", square(7));
    return 0;
}

int square(int x)
{
    return x * x;
}

Compile + run.

Worked? Prints 49. Try removing the prototype line — observe the compile error.

2. 12-strings.c — strings are char arrays

What: in C, a string is just an array of characters ending with the null character \0.

Create. Type:

#include <stdio.h>

int main(void)
{
    char name[] = "Dave";
    printf("name: %s\n", name);
    int i = 0;

    }
    return 0;
}

Pointers

What's a pointer? A pointer is a variable that holds the address of another variable. Two operators: &x means "the address of x", *p means "the value at the address p points to". Once you grok these two, pointers click.
flowchart LR A["int x = 42
(at address 0x1A)"] --> B["int *p = &x
(p holds 0x1A)"] B -->|"*p reads x"| A

1. 14-pointer-basic.c — address and dereference

Create. Type:

#include <stdio.h>

int main(void)
{
    int x = 42;
    int *p = &x;
    printf("address of x: %p\n", (void*)&x);
    printf("p holds:      %p\n", (void*)p);
    printf("*p (value):   %d\n", *p);
    return 0;
}

Compile + run.

Worked? First two addresses are equal; *p is 42.

2. 15-pointer-modify.c — change a value through a pointer

Create. Type:

#include <stdio.h>

void increment(int *n)
{
    (*n)++;
}

int main(void)
{
    int x = 10;
    printf("before: %d\n", x);
    increment(&x);
    printf("after:  %d\n", x);
    return 0;
}

Compile + run.

Worked? before: 10, after: 11. The function modified x via the pointer.

Heap memory: malloc + free

The C rule Every malloc needs exactly one free. Forget to free → memory leak. Free twice → crash. This is why memory bugs are common in C and why higher-level languages exist.

1. 16-malloc.c — allocate, use, free

Create. Type:

#include <stdio.h>
#include <stdlib.h>

int main(void)
{
    int *arr = malloc(5 * sizeof(int));
    if (arr == NULL) return 1;
    for (int i = 0; i < 5; i++) arr[i] = i * 10;
    for (int i = 0; i < 5; i++) printf("%d ", arr[i]);
    printf("\n");
    free(arr);
    return 0;
}

Compile + run.

Worked? Prints 0 10 20 30 40. The memory was on the heap; free returned it.

Caesar cipher: caesar.c

A classic CS50 exercise. Read a string and a key from command-line arguments, shift every letter by the key.

1. Create caesar.c

Type:

#include <ctype.h>
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

int main(int argc, char *argv[])
{
    if (argc != 3) {
        printf("Usage: caesar <text> <key>\n");
        return 1;
    }
    char *text = argv[1];
    int key = atoi(argv[2]);
    for (int i = 0, n = strlen(text); i < n; i++) {
        char c = text[i];
        if (isupper(c))      putchar('A' + (c - 'A' + key) % 26);
        else if (islower(c)) putchar('a' + (c - 'a' + key) % 26);
        else                 putchar(c);
    }
    putchar('\n');
    return 0;
}

2. Compile + test

gcc -Wall -Wextra caesar.c -o caesar
./caesar HELLO 3
./caesar "World" 13
Worked? First prints KHOOR, second prints Jbeyq.
