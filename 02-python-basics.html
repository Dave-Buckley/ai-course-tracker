3. Create a virtual environment (venv)

Mini-project: unique words + most common

1. Add unique-words count

In wordcount.py, add this function above the if __name__ ... block:

def unique_words(s: str) -> int:
    return len(set(s.lower().split()))

2. Add a most-common function (no Counter — write the loop)

Why no Counter: writing the dict-building loop yourself teaches the pattern. collections.Counter is fine in real code, but seeing it from scratch makes it click.

Add:

def most_common(s: str) -> str:
    counts = {}
    for word in s.lower().split():
        counts[word] = counts.get(word, 0) + 1
    best_word = ""
    best_count = 0
    for word, count in counts.items():
        if count > best_count:
            best_word = word
            best_count = count
    return best_word

3. Use them in __main__

Update the if __name__ == "__main__": block:

if __name__ == "__main__":
    text = "the quick brown fox jumps over the lazy dog the"
    print(f"chars: {count_chars(text)}")
    print(f"words: {count_words(text)}")
    print(f"unique: {unique_words(text)}")
    print(f"most common: {most_common(text)}")

Save. Run.

Worked? Most common word is the (appears twice).
